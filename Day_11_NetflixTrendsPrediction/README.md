# Day 11: Netflix Trends Regression Prediction

Welcome to Day 11 of your AI journey! This script uses regression to analyze trends in Netflix content, predicting whether the proportion of TV shows versus movies increases over time, with forecasts for future years.

## Learning Keys
- Basics of linear regression with Scikit-learn
- Data preprocessing and trend analysis with Pandas
- Visualizing predictions with Matplotlib

## Requirements
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- KaggleHub
